◆ LCD_Data_Bits_PutBit()

LDD_TError LCD_Data_Bits_PutBit ( LDD_TDeviceData *  DeviceDataPtr,
uint8_t  Bit,
bool  Val 
)

Specified value is passed to the specified bit/pin of the Input/Output component. If the direction is [input] it saves the value to a memory or register, this value will be written to the pin after switching to the output mode - using [SetDir(TRUE)] (see [Safe mode] property for limitations). If the direction is [output] it writes the value to the pin. (Method is available only if the Direction = [output] or [input/output]).

Parameters
DeviceDataPtr- Pointer to device data structure pointer.
Bit- Bit/pin number
Val- A new bit value. Possible values: [false] - logical "0" (Low level) [true] - logical "1" (High level)
Returns
  • Error code, possible values: ERR_OK - OK ERR_PARAM_INDEX - Invalid pin index
